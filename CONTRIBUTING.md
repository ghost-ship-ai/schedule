# Contributing

This repository is maintained autonomously by [Ghost Ship AI](https://github.com/ghost-ship-ai). There are no human committers.

## Reporting Bugs

Open an issue using the **Bug Report** template. Include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (browser, OS, runtime version)

An automated agent will respond within minutes, classify the issue, and post a plan if a fix is needed.

## Requesting Features

Open an issue using the **Feature Request** template. Describe:

- What you'd like to see
- Why it's useful
- Any implementation ideas (optional)

Features that fit the project's scope will be planned and implemented. Out-of-scope requests will be explained and closed.

## Pull Requests

Community pull requests are welcome. However, to maintain the bot-only commit history:

1. Open a PR with your changes
2. An automated reviewer will review your code
3. If approved, your changes will be **re-applied under the bot identity** — your original PR will be closed with credit
4. You will be credited as `Co-authored-by` in the commit message

This ensures the commit history remains fully automated while acknowledging your contribution.

## Security

Please see [SECURITY.md](SECURITY.md) for reporting security vulnerabilities. Do **not** open public issues for security concerns.

## Code of Conduct

Be respectful. This project is maintained by automation, but the humans who use it deserve a welcoming environment.
